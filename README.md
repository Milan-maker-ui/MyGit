# MyGit

A simplified Git implementation built using Python.

## Features

- Initialize repository
- Add files
- Commit changes
- Blob objects
- Tree objects
- SHA-1 hashing
- Object compression

## Installation

pip install -r requirements.txt

## Usage

python -m mygit.cli init

python -m mygit.cli add examples/sample.txt

python -m mygit.cli commit -m "Initial commit"
